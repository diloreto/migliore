== Description ==

WP Pixabay helps you search millions of free photos, vectors and art illustrations from pixabay.com then insert into content or set as featured image very quickly.

= Features =

- Search images with powerful filters
- Quick insert image with original URL
- Upload image(s) to Media Library (Premium Version)
- Use image as featured image
- Multiple select support
- WPML integration

= Translators =

Available Languages

- English (Default)

If you have created your own language pack, or have an update for an existing one, you can send [gettext PO and MO file](http://codex.wordpress.org/Translating_WordPress "Translating WordPress") to [us](https://wpclever.net/contact "WPclever.net") so we can bundle it into WP Pixabay.

= Need support? =

Visit [plugin documentation website](https://wpclever.net "plugin documentation")

== Installation ==

1. Unzip the wp-pixabay.zip
2. Copy wp-pixabay folder to wp-content/plugins
3. Go to Installed Plugins, find WP Pixabay and click Active
4. Now when you create/edit the post, you can see the Pixabay button above the editor
5. Enjoy!
